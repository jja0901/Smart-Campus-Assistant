#  Smart Campus Assistant

A **Java-based console application** that helps manage campus-related information, including students, courses, books, and events all in one place.  
This project demonstrates **object-oriented programming (OOP)** concepts like encapsulation, composition, and modular design.

---

##  Overview

The Smart Campus Assistant is a text-based system designed to simulate features of a universityâ€™s internal information assistant based on the specifications given for this project.
It lets users perform operations like adding students, assigning grades, viewing courses, managing library books, and scheduling events.

---

##  Project Structure

| File | Description |
|------|--------------|
| **SmartCampusAssistant.java** | The main driver of the program. Runs menus, handles user input, and connects all modules. |
| **Student.java** | Defines the `Student` class with ID, name, and course grades. Includes a nested `CourseGrade` class. |
| **Course.java** | Represents a campus course, including course code, name, and instructor. |
| **Book.java** | Models library books with title, author, and availability tracking. |
| **Event.java** | Manages campus event information. |

---

##  Features

 - Add and view students, courses, and books  
 - Assign grades to students and calculate GPA  
 - Manage book checkouts and returns  
 - Create and view campus events  
 - Built-in calculator and utility tools
 - Console-driven interface
